# 🎂 Dedicatoria de Cumpleaños · Para Mamá

Una experiencia interactiva y animada que combina pastel, velitas que se apagan al tocarlas, fotos, flores y mensajes dedicados.

---

## 📂 ¿Qué hay en esta carpeta?

```
birthday/
├── index.html          ← La página web completa
├── images/             ← Todas las imágenes (pastel, flores, fotos)
└── README.md           ← Este archivo (instrucciones)
```

## 🌐 Cómo enviarla por WhatsApp (3 minutos)

WhatsApp no puede mostrar una web directamente — necesita un **link**. La opción más rápida y gratis:

### Opción A · Netlify Drop (recomendada, sin registro)

1. Abre **https://app.netlify.com/drop** en tu navegador
2. **Arrastra la carpeta `birthday` completa** dentro del cuadro grande de la página
3. Netlify te dará un link tipo `https://nombre-random-123.netlify.app` en unos segundos
4. Copia ese link y envíalo por WhatsApp a tu mamá 💌

### Opción B · GitHub Pages

1. Crea una cuenta en https://github.com
2. Crea un repositorio nuevo
3. Sube el contenido de esta carpeta
4. En *Settings → Pages*, activa "GitHub Pages" desde la rama main
5. Te dará un link `https://tu-usuario.github.io/nombre-repo/`

### Opción C · Si solo quieres probarla en tu compu

Doble click en `index.html` y se abrirá en tu navegador.

## 💡 Cómo funciona la experiencia

Tu mamá verá 6 "escenas" que avanzan tocando botones:

1. **🌙 Apertura** — Cielo estrellado con su nombre
2. **🎂 Pastel** — Las velitas se encienden una a una. Ella toca cada velita para apagarla (con humo). Al apagar todas → fuegos artificiales y "tu deseo se hará realidad"
3. **💌 Dedicatoria** — Mensaje escribiéndose letra por letra con su foto
4. **📸 Recuerdos** — Galería con sus 3 fotos y frases dedicadas
5. **💐 Flores** — Los dos ramos aparecen flotando desde abajo
6. **❤️ Final** — Mensaje de cierre con corazones flotando

**Bonus:** Botón ♪ arriba a la derecha → reproduce "Happy Birthday" en bucle.

## ✏️ ¿Qué partes quieres personalizar?

Dime cualquiera de estas cosas y te las cambio:

- **Nombre real** de tu mamá (en vez de "mamá")
- **Mensaje principal** de la dedicatoria (Escena 3)
- **Frases de cada foto** en la galería (Escena 4)
- **Mensaje final** (Escena 6)
- **Edad / número** de cumpleaños (puedo agregar "tus 60 años")
- **Más flores, más fotos**, otro pastel, etc.
- **Cambiar colores**, otra tipografía, otro estilo
- **Otra música** (mariachi, Las Mañanitas, balada, etc.)

Solo dime qué cambiar y te paso una versión actualizada 🛠️
